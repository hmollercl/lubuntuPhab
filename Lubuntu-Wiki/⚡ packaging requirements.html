Purpose of this document

This is meant to provide all the requirements necessary to follow the Packaging Tutorial.

Requirements

  1. Software
    1. apt install ssh git php-curl php-cli devscripts tar quilt
    2. Follow "Installing Arcanist" here.
  2. Configuration
    1. Uncomment the deb-src lines in /etc/apt/sources.list so apt build-dep works correctly.
    2. So you get credit where credit's due:
      1. For git:
        • git config --global user.name "your full name"
        • git config --global user.email "your email address"
      2. For Debian tools, add the following to $HOME/.bashrc:
        • export DEBFULLNAME="your full name"
        • export DEBMAIL="your email address"
        • export EMAIL="your email address"
    3. Set up ssh:
      • Easy way
        1. ssh-keygen
        2. Accept the default name ($HOME/.ssh/id_rsa)
        3. Enter a password ≥ 5 characters
      • Hard way
        1. ssh-keygen -C some-identifying-info -t ed25519 # more secure
        2. Select a unique name e.g. $HOME/.ssh/phab # allows you to have more than one key
        3. Pick an even better password!
        4. Edit $HOME/.ssh/config to include
          • Host phab.lubuntu.me
          • IdentityFile ~/.ssh/phab
    4. Add ssh key to Phab:
      1. Go to Settings
      2. Click on "SSH Public Keys" under "Authentication" on the left side
      3. Click SSH Key Actions → Upload Public Key
      4. Give your key a name (anything works) and then copy and paste the public key from the step above
    5. $HOME/.quiltrc from packaging guide
    6. Final Arcanist tweaks:
      1. arc set-config phabricator.uri "https://phab.lubuntu.me/"
      2. arc install-certificate

Extras

  1. ssh agent with keychain TODO
  2. gpg setup
    1. Get the software: sudo apt install gpg
    2. Make a new key: gpg --full-generate-key
      1. Select an RSA and RSA pair
      2. Select 4096 bits for key size
      3. Select expiration date, or none (note you can always move the date out farther)
      4. Verify and accept
      5. Enter your real name
      6. Enter your email address
      7. A comment is not necessary
      8. Give it a good password
      9. It takes a while to gain enough entropy to finish the creation. Open a new terminal and find / 2>/dev/null and that should help it along
