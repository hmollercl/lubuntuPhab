Description

I could reproduce this when setting my resolution in a VM insanely low, but I just got a report from someone using an NVIDIA graphics card with the open source driver that SDDM doesn't let them click the password box to login. They had to switch to tty2 and install the NVIDIA drivers and reboot.

We need to test this and see if we can blame upstream SDDM or if it's a problem with our QML.
