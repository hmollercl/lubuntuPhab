Description

We have Telegram everywhere, so I think we're set on that front. Heck, we even have direct links. Here's what we need:

  • Remove the Telegram link from the "We are social" section of the links page.
  • Rewrite the IRC section to be more chat-general
  • Move Telegram links into the above section
  • While we're at it, move in Matrix

We currently only have approval to have the Telegram bridge (TeleIRC) on #lubuntu-devel. The unfortunate thing is that we publish this so often have offtopic and/or support questions showing up there. It would be nice if we had supergroups for all the channels so we could better direct traffic and keep things on top.

Currently, Unit193 finds the Telegram bridge confusing and there seems to be some consensus among ops that it may not be appropriate for unlogged channels (like offtopic).

At time of writing, I know an email has been sent to the IRC Council mailing list.
